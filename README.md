# Data Jobs Dashboard w/ Power BI

![Dashboard Page 1](/Dashboard_01_page_01.png)

## Introduction

This project presents an interactive **Power BI Dashboard** that explores the global data job market using a comprehensive dataset of job postings. It helps users analyze hiring trends, salary insights, remote work opportunities, and job requirements across different countries and data-related roles.

The dashboard transforms raw data into meaningful visualizations, allowing users to quickly identify patterns and gain valuable insights through an intuitive and interactive reporting experience.

---

## Skills Showcased

- **Microsoft Power BI** – Dashboard creation and data visualization
- **Power Query** – Data cleaning and transformation (ETL)
- **Data Modeling** – Building relationships between tables
- **DAX (Data Analysis Expressions)** – Creating calculated columns and measures
- **Interactive Dashboards**
  - Slicers

- **Data Analysis**
  - Identifying trends

## Dashboard Overview

### Page 1: High-Level Market View

![Dashboard Page 1](/Dashboard_01_page_01.png)

The main dashboard provides an overview of the global data job market, allowing users to explore key hiring and salary insights through interactive visuals.

**Key Highlights:**

- Total job postings and overall job market statistics
- Median yearly and hourly salary analysis
- Monthly job posting trends
- Salary comparison across different job titles
- Relationship between yearly and hourly salaries
- Interactive filters for dynamic data exploration

---

### Page 2: Job Title Drill-Through

![Dashboard Page 2](/Dashboard_01_page_02.png)

The drill-through page provides a detailed analysis of an individual job role, helping users better understand specific career opportunities.

**Key Highlights:**

- Median yearly and hourly salary for the selected role
- Work-from-home availability
- Degree requirements
- Health insurance availability
- Global distribution of job locations
- Most popular job posting platforms
- Employment type distribution
